1. font-family

font - family adalah format untuk mengubah jenis font, font - family memiliki 2 value yaitu (nama font dan generic family)

2. font-size

untuk mengatur ukuran dari font dan memiliki 3 value (px, %, em)

3. font-weight

untuk mengatur ketebalan font dan memiliki 5 value (lighter, normal,100-900, bold, bolder

4. Font-Variant

font variant adalah mengubah font menjadi small caps dan normal (untuk setiap huruf awalan dia besar dan huruf selanjutnya kecil tanpa harus menggunakan font-size

5. Font-Style

mengubah bentuk font bercetak miring dan memiliki 3 valaue yaitu (normal, italic, dan oblique)

6. Line-Height

line - height untuk mengatur spasi antar baris, dan ini memiliki 3 value yaitu (normal, px, em)
