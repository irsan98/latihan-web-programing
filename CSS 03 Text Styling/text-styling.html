kali ini kita akan mempelajari Text-Styling, properti pada Text-Styling terbagi menjadi beberapa bagian yaitu :

  1. Color = untuk memberikan warna pada tulisan
  2. text-align = untuk memberikan format paragraf/text
  3. text-indent = untuk identasi pada paragraf
  4. text-decoration = mengatur dekorasi pada text
  5. text-transform = mengubah jenis huruf besar, kecil/kapital
